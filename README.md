# 📰 News API App

A modern and responsive News Application built using *HTML, CSS, and JavaScript*.  
The app fetches real-time news data using a News API and displays it in a clean, user-friendly interface.

## 🚀 Features

- 🔥 Top Headlines section
- 📂 Category-based news filtering
- 🔍 Search functionality
- ❤️ Save to Favorites
- 🌍 Country & language filters
- 📱 Fully responsive design
- ⚡ Dynamic API data rendering

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript (ES6)
- News API

## 📸 UI Sections

- Home (Top Headlines)
- Categories
- Search
- Favorites

## 📦 How to Run Locally

1. Clone the repository
2. Add your News API key in script.js
3. Open index.html in your browser
